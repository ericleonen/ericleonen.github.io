Ideas are cheap, so I'm not sure if any of these are possible/useful. But they sure sound cool...

- **Automated Causal Inference Agent:** An agent that accepts a dataset, documentation, and prompts that helps a researcher perform causal inference studies.
- **Logical Fallacy Goggles:** An extension that uses AI to flag common logical fallacies used in online debates (i.e. ad hominem, straw man, slippery slope). Would be interesting to learn the distribution of fallacies on different online spaces like Stack Overflow, X, or Reddit.
- **Dumb Lucking Money:** Enter a stock and find obviously wrong signals strongly correlated with it
- **Kamibase:** Database of origami crease patterns. Main features are a semantically searchable database of patterns, an AI-powered image-to-pattern tool, and an intuitive tool for drawing patterns online.
- **Inspirational Running App:** Running app that not only tracks your runs (duration, length, pace, etc.) but plays inspirational sound bites at *key* moments (e.g. you slow down, you're almost done)
