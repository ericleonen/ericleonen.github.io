Fully automated, LLM-powered job board for predoctoral researcher position openings, meant to solve the issue that NBER, PREDOC.org, etc. list a lot of links, but lack organization in terms of important dates (application deadlines or start dates). Used terminal-based [Claude Code](https://code.claude.com/docs/en/overview) for the first time, [Neon](https://neon.com/) to host my DB, [Vercel Functions](https://vercel.com/docs/functions) to host my API, and [GitHub Actions](https://github.com/features/actions) to run my scripts daily.
